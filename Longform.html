Welcome To Run An Empire

Thanks very much for agreeing to take part in the beta, you should be up and running in a few minutes. Just follow the below steps to install the game via Apple's TestFlight application.

Installing the beta

1. Click the “Open in TestFlight” button in the invitation email you should have received from testflight. (you will need to do this in the default “Mail” app on your iPhone. It doesn’t seem to work in the Gmail app for iOS, for example. If you're using a third-party app, click the button in the email, copy the redeem code and then go to TestFlight and hit 'redeem' and paste in that code.)

2. The link will open the TestFlight app page on the App Store - tap the “Get” button.

3. Once it’s downloaded, tap the “Open” button.

4. Agree to the Terms of Service.

5. In the TestFlight app, Run An Empire should appear. Tap the “Install” button.

6. You should be all set - open the Run An Empire app and create an account.

What to expect

The game you are about to play continues to be a work in progress. There will be new features on a regular basis and no doubt the occasional bug. We’d appreciate your help in identifying these - if something doesn’t work, happens unexpectedly, or doesn’t make sense then please tell us. When identifying bugs, detail is everything - so please try to explain what you did beforehand in as much detail as possible.

To report a bug, please use the ‘Send feedback’ button built into TestFlight - or check our Beta Help Centre for known issues. Screenshots are always welcome, too!

You should also be aware that your beta empire may not be brought forward into the fully launched version of the game and towns aren’t automatically activated in the game - we add them one at a time. If you attempt to play away from home and receive a message saying “your current location has not yet been added to the game”, this is the reason why but let us know and we can add it.
